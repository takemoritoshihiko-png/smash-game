# SMASH GAME - LAN対戦

## LAN対戦の起動方法

### 必要なもの
- Node.js（v14以上）
- 同じLAN（Wi-Fi / 有線LAN）に接続された2台のPC

### サーバー起動

```bash
cd smash-game
node server.js
```

起動すると以下が表示されます：
```
========================================
  SMASH GAME - LAN Server
========================================
  Local:   http://localhost:3000
  LAN:     http://192.168.x.x:3000

  Share the LAN address with your opponent!
========================================
```

### 接続方法

#### ホスト（サーバー側）
1. `node server.js` でサーバーを起動
2. ブラウザで `http://localhost:3000` を開く
3. タイトル画面で **Lキー** を押す
4. 「ホスト（サーバー）」を選んでEnter
5. 対戦相手の接続を待つ

#### ゲスト（接続側）
1. ブラウザでホストのLANアドレス（例: `http://192.168.1.10:3000`）を開く
2. タイトル画面で **Lキー** を押す
3. 「ゲスト（接続）」を選んでEnter
4. IPアドレスを入力（表示されているアドレスと同じなら `localhost:3000` でもOK）
5. Enterで接続

### LAN対戦の操作
両プレイヤーとも同じ操作キーを使用：

| キー | アクション |
|------|-----------|
| ← → | 移動 |
| ↑ | ジャンプ |
| ↓ | しゃがみ |
| Z | 通常攻撃 / コマンド技 |
| X | 強攻撃 / 投げ |
| C | 必殺技 |

### 注意事項
- 外部パッケージ不要（`npm install` 不要）
- サーバーはポート3000を使用
- ファイアウォールでポート3000を許可する必要がある場合があります
- 接続が切れた場合「切断されました」と表示され、Enterでタイトルに戻ります
